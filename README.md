# Meta Ads Performance Analysis Dashboard
Power BI | SQL | DAX | Marketing Analytics
This project presents a comprehensive Power BI dashboard to analyze the performance of Meta (Facebook & Instagram) advertising campaigns. 
It provides deep insights into impressions, clicks, engagement, conversions, audience demographics, budget allocation, and campaign effectiveness to 
help improve ad strategy and ROI.

# Project Overview
The Meta Ads Performance Dashboard is designed to help marketers and analysts understand how campaigns perform across different 
audience groups, platforms, ad types, and time periods. It includes multiple interactive report pages for Facebook and Instagram,
along with a calendar heatmap to identify weekly and monthly performance trends.

# Tools & Technologies Used:
Power BI Desktop
SQL Server
DAX (Data Analysis Expressions)
Power Query
Excel

# Key Features
1. KPI Summary Cards
Impressions, Clicks, Shares, Comments
Engagement Rate, CTR, Conversion Rate, Purchase Rate
Total Budget & Average Budget per Campaign

2. Audience Insights
Gender distribution (Male/Female/Other)
Age-wise impressions
User interests (Finance, Fashion, Travel, Gaming, etc.)

3. Campaign Performance
Ad Type Analysis (Image, Video, Carousel, Stories)
Country-wise impressions
Duration vs Impressions scatter plot
Campaigns ranked by Purchase Rate

4. Trend Analysis
Month-wise impression trends
Day-wise and Time-period performance
Weekly calendar heatmap

5. Multi-Page Dashboard
Facebook Report 1
Facebook Report 2
Instagram Report 1
Instagram Report 2
Calendar Heatmap

6. Dynamic KPI Measure Selector
Switch between KPIs like Impressions, Clicks, Engagement, CTR, Budget, etc.

# Data Preparation & Modeling
Cleaned and transformed raw Meta ads dataset using Power Query
Built a star schema data model with fact and dimension tables
Created DAX measures for KPIs:
CTR
Conversion Rate
Engagement Rate
Purchase Rate
Budget metrics
Validated campaign-level insights using SQL queries

# Business Impact
Identified top-performing campaigns and low-performing ones
Analyzed ideal posting times and audience segments
Provided insights to improve ad spend efficiency
Helped understand which creatives and regions deliver maximum returns
